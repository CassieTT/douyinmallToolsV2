# 抖音商城联合 Logo 生成器

打开 `index.html` 即可使用。工具支持双品牌和多品牌模式，仅接受 PNG 上传，并可分别下载彩色与反白的 4x PNG 透明底图片。

生成规则：

- 双品牌模式使用 “|” 连接，上传 logo 固定在分割线右侧。
- 多品牌模式不加分割线，品牌之间保持等距。
- 双品牌模式可选择抖音商城 logo 居前或居后。
- 多品牌模式支持多次选择 PNG 追加上传，并可在“呈现顺序”中拖动抖音商城与各品牌 logo 的排列顺序。
- 反白版本可选择“制作反白logo”自动转白，或选择“上传反白logo”按品牌顺序提供独立反白 PNG。
- 上传 logo 高度最高限制为抖音商城 logo 高度的 1.5 倍。
- 可微调上传 logo 高度、垂直位置和整体边距。
